#next-lang-decimal
##description:
+ A decimal lib based on next toolkit.

##usage:
+ Get Decimal util:
```javascript
  var Decimal=nx.lang.Decimal;
```
+ Add:
```javascript
  Decimal.add(0.1,0.2,0.3);
```

+ Sub:
```javascript
  Decimal.sub(0.3,0.1);
```

+ mul:
```javascript
  Decimal.mul(0.3,0.1,0.2);
```

+ div:
```javascript
  Decimal.mul(0.3,0.1);
```
